# Pasta de Assets

## Logo do Sistema

Coloque a logo do BarberCash nesta pasta com o nome: **`logo.png`**

### Formatos aceitos:
- PNG (recomendado)
- JPG/JPEG
- SVG
- WEBP

### Tamanho recomendado:
- Mínimo: 200x200 pixels
- Ideal: 512x512 pixels
- Formato: Quadrado (1:1)

Após adicionar a logo, o sistema irá carregá-la automaticamente.
